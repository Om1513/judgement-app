# Judgement

A cross-platform mobile and web application built with Expo.

## Getting Started

### Prerequisites

- Node.js (v18 or newer)
- npm or yarn
- Expo Go app on your mobile device (for testing on physical devices)

### Installation

```bash
npm install
```

### Running the App

```bash
# Start the development server
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios

# Run on Web
npm run web
```

## Project Structure

```
judgement/
├── App.js          # Main application entry point
├── app.json        # Expo configuration
├── assets/         # Static assets (images, fonts, etc.)
├── index.js        # App registry
└── package.json    # Dependencies and scripts
```

## Platforms

- **Mobile**: iOS and Android via Expo Go
- **Web**: Browser support via react-native-web
